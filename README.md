# Physics for Class

物理授業用WEBアプリケーション

> 最終更新: 2026-01-14 20:06

## 概要

このアプリは物理学習をゲーミフィケーションの要素を取り入れて楽しく効率的に行うためのツールです。

## 機能

- 📚 **問題演習** - 難易度別（初級・中級・上級・ALL）の問題に挑戦
- 📊 **成績管理** - 正答率、EXP、ストリークなどの詳細な記録
- 🏆 **ランキング** - 他のユーザーとの競争
- 🎨 **テーマカスタマイズ** - 50種類以上のビジュアルテーマ
- ☁️ **クラウド同期** - Google Apps Script連携でデータを保存
- 📈 **パフォーマンス分析** - グラフによる学習進捗の可視化

## 技術スタック

- HTML5 / CSS3 / JavaScript
- TailwindCSS
- Chart.js（グラフ表示）
- KaTeX（数式レンダリング）
- Google Apps Script（バックエンド）

## 使い方

1. ブラウザで `physics1.0.html` を開く
2. ユーザー名を登録
3. 問題セッションを選択して学習開始
